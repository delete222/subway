# Subway Commute PWA

一个面向昌平线通勤的 PWA 页面，重点展示南邵和西二旗之间的地铁与班车信息。

## 功能

- 自动判断工作日和休息日。
- 12:00 前默认展示上班方向，12:00 后默认展示下班方向。
- 09:35 到 17:50 默认展示班车页，其余时间默认展示地铁页。
- 上班地铁优先关注昌平东关发车、南邵上车的车次。
- 下班地铁优先关注可到达南邵的车次。

## 本地开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```

## 部署

项目通过 GitHub Pages 发布，推送到 `main` 分支后由 GitHub Actions 构建部署。
