# 地铁班次与班车查询H5页面需求文档

## 1. 主题与简介
地铁班次与班车查询工具,帮助用户快速查看可赶上的地铁班次和公司班车班次,并提供实时倒计时功能。支持上班和下班两种场景的智能班次推荐,集成地图定位服务,提供精准的出行建议。用户可通过Tab切换查看地铁倒计时或班车倒计时。

## 2. 核心功能
### 2.1 时间显示
- 显示当前实时时间
- 显示距离下一班可赶上地铁/班车的倒计时
- 支持手动选择班次倒计时:点击任意时间点,倒计时切换为该班次的倒计时
- 提供恢复默认功能:一键恢复到自动推荐的下一班可赶上地铁/班车的倒计时

### 2.2 上班班次查询
- 地铁班次查询模块:展示昌平东关站发车班次(重点关注,空座较多)和昌平西山口发车班次(全站运行),通过色块区分两个发车站点
- 班车班次查询模块:提供方舟→百度大厦和百度大厦→方舟两条线路的选择,用户选择线路后展示最近的班车时间和倒计时
- 通过Tab切换:用户可切换查看地铁倒计时或班车倒计时

### 2.3 下班地铁功能模块
- 时段识别:7-9点为上班时段,18-20点为下班时段
- 智能筛选:仅推荐能到达预设家庭地址的地铁线路
- 过滤无法到达的班次
- 班次时间点支持点击交互,点击后倒计时切换为该班次

### 2.4 地图服务集成
- 接入地图API实现定位功能
- 自动计算当前位置到西二旗地铁站的步行时间
- 显示具体步行分钟数
- 计算当前位置到达北京昌平南邵站C口的步行距离
- 根据步行距离测算出门时间建议

### 2.5 智能乘车建议系统
- 实时分析当前时间点可乘坐的班次
- 明确标注禁止乘坐的班次
- 输出格式:
  - 当前位置:XXX(根据定位显示)
  - 步行到西二旗地铁站需XX分钟
  - 推荐乘坐班次:18:02/18:05/18:10...(绿色标识)
  - 禁止乘坐班次:18:00/18:07/18:15...(红色警示)

### 2.6 倒计时控制功能
- 默认模式:自动显示距离下一班可赶上地铁/班车的倒计时
- 手动选择模式:点击任意班次时间点,倒计时切换为该班次的倒计时
- 恢复默认按钮:提供〔恢复默认〕或〔恢复正常〕选项,一键回到自动推荐模式

### 2.7 班车线路选择
- 提供两条班车线路:
  - 方舟→百度大厦
  - 百度大厦→方舟
- 用户选择线路后,展示该线路的最近班车时间和倒计时

## 4. 设计风格
简洁实用的移动端界面,重点突出时间信息和班次区分。昌平东关站发车班次采用醒目色块标识,推荐乘坐班次使用绿色标识,禁止乘坐班次使用红色警示标识。班次时间点支持点击交互,选中状态有明显视觉反馈,恢复默认按钮位置醒目易操作。Tab切换按钮清晰展示地铁和班车选项,班车线路选择界面简洁明了。
